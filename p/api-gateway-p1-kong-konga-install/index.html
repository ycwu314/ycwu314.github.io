放开管理端口

默认端口：

  • Proxy 8000：接收客户端的 HTTP 请求，并转发到后端的 Upstream。
  • Proxy 8443：接收客户端的 HTTPS 请求，并转发到后端的 Upstream。
  • Admin 8001：接收管理员的 HTTP 请求，进行 Kong 的管理。
  • Admin 8444：接收管理员的 HTTPS 请求，进行 Kong 的管理。
