CorsConfiguration

CorsConfiguration 是配置类。

当允许的 Origin 为 *，且 credentials 为true，则返回当前请求的 Origin、而不是*。

CorsFilter

CorsProcessor 是 CORS 处理器抽象，DefaultCorsProcessor。
CorsFilter 是 CORS 的过滤器，调用 CorsProcessor 进行实际操作。

在 w3c 标准中，浏览器对复杂跨域请求，首先发送 options 请求判断能否执行。
对应的是CorsUtils#isPreFlightRequest()
