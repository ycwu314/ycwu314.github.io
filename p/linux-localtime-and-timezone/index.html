折腾 alpine，遇到时区问题，于是整理 linux 的 localtime 和 timezone。

localtime vs timezone

/etc/localtime 是用来描述本机时间，而 /etc/timezone 是用来描述本机所属的时区。
修改时区

localtime 影响 linux date 命令。
timezone 影响 java Date 类型。

修改时间，建议使用软链接方式：

java TimeZone

上面提到 /etc/timezone 影响 java 的 Date 行为。java 获取 timezone 的优先级，参见 TimeZone.java：

  1. 用户指定的 user.timezone
  2. 系统的 timezone，通过 navtive 方法获取。访问 TZ 环境变量、/etc/timezone等
  3. GMT

关键是 findJavaTZ_md()，是一个平台相关的实现。
打开一个看看：/src/solaris/native/java/util/TimeZone_md.c

zdump 命令

Zdump 对命令行中的每一个 zonename 输出其当前时间。

tzselect 命令

tzselect 提供交互引导方式设置时区。
